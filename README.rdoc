= Form-San

Form-San is a lean extension for the Rails form builder.

      <% f.fields do %>

            <p class="hint">Shown on the site.</p>
          <% end %>
        <% end %>
      <% end %>
    <% end %>
    
    <div class="submit">
      <%= f.submit 'Sign up' %>
      <%= link_to 'Cancel', root_path, :class => 'cancel' %>
    </div>
  <% end %>
